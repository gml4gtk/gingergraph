    /angle dy dx atan def

    /base arrowlength headlength sub def

    angle rotate

    0 0 moveto
    base 0 lineto
    stroke

    newpath
    base halfheadthickness moveto
    arrowlength 0 lineto
    base halfheadthickness neg lineto

   end
